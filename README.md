# ChallengeBackGokoshiJr

This is the Challenge Back repository.

## Install

```git
$ git clone https://github.com/GokoshiJr/ChallengeBackGokoshiJr.git
```
```bash
npm install
```

## Create .env file
```
touch .env
```
### Add DB_URI variable in .env file
DB_URI = YOUR_DB_URI

## Run the app
```
node index.js
```
